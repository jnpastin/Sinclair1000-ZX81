# Hardware Documentation

This directory contains all hardware-related documentation for the Z80 Validation test rig.

## Contents

### Schematics
- **Z80_TestRig.kicad_sch** - Main schematic showing Z80 to Arduino Mega 2560 connections
- **Z80_TestRig.kicad_pro** - KiCad project file
- **Z80_TestRig.pdf** - PDF export of schematic for easy viewing

### Pin Mapping
- **pin_mapping.md** - Detailed pin-to-pin connections between Z80 and Arduino
- **pin_mapping.csv** - Machine-readable pin mapping table

### Physical Design
- **perfboard_layout.md** - Layout diagram for perfboard construction
- **wiring_diagram.pdf** - Wiring diagram showing all connections
- **photos/** - Photos of the completed test rig

### Datasheets
- **Z80_datasheet.pdf** - Z80 CPU datasheet
- **Arduino_Mega_pinout.pdf** - Arduino Mega 2560 pinout reference

### Bill of Materials
- **BOM.md** - Complete parts list with quantities and sources

## Notes
- All KiCad files are version 7.0+ format
- Schematics use standard Z80 and Arduino Mega symbols
- Pin numbering follows IC datasheet conventions
